Assignment 1: Program A Microcontroller To Do Something

My idea is to program the microcontroller to make a motor go faster or slower using a potentiometer.

Since my motor for my kinetic sculpture broke, I got a new hobby motor from the Fabrication Storage Compartment and an Arduino Uno. (I was going to use ESP 32 but it was too difficult for me to use.) I got some wires, a potentiometer, and an L9110 Motor Driver and started assembling.

I started by plugging a wire into GND (ground) and into 3.3V and attached it to GND and VCC respectively into the L9110 Motor Driver. Then, I screwed my motor into the L9110 Motor Driver and attached my potentiometer to A0, A2, and A4 on the Analog IN section on my Arduino Uno. Next, I plugged some wires into pins ~3 and 4 and inserted them in A-1A and A-1B respectively into the L9110 Motor Driver as well; my circuit is now ready and I can move onto programming.

Time for programming! I went into the programming examples on the professor's website and picked the code that allowed me to change the speed of my motor using a potentiometer. I then placed that code into the Arduino IDE; the code is shown below. 

  /*  Code to run motor in one direction based on potentiometer reading.
 *  Motor driver speed control on pin 3 (direction LOW by default)
 *  Potentiometer plugged into A0, A2, and A4 on Arduino Uno (or similar).
 */

void setup() {
  pinMode(3, OUTPUT);
  pinMode(A0, OUTPUT);  //GND
  pinMode(A4, OUTPUT);  //3.3V

  digitalWrite(A0, LOW);
  digitalWrite(A4, HIGH);
}

void loop() {
  int pot_value = analogRead(A2);   //pot wiper on A2
  int motor_speed = map(pot_value, 0, 1023, 255, 0); // map motor_level to pot_value so zero corresponds to lowest speed
  analogWrite(3, motor_speed);
  delay(1);
}

My project worked!

Assignment 2: Draw A Schematic Of My Circuit

I used Circuit Diagram to make my circuit. Circuit Diagram does not have a L9110 Motor Driver component so I used another component and went with it. I added my arduino Uno in and hooked a potentiometer to the A0, A2, and A4 pins. Then, I added two wires going from GND and 3.3V to GND and VCC respectively. I then added two pins from ~3 and 4 to LE (A-1A) and CO (A-1B). Finally, I added my motor and plugged my wires from LE and CO to the motor and I got my circuit diagram!
