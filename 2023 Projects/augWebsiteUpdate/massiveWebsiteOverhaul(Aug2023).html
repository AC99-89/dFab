Goals:
Make An Attractive Website
Make The Website Mobile Friendly

The images below shows you how the website looked from August 7 to August 24.



The images below shows you how the website looks now.

How I Updated The Website

Step 1: Get Resources And Help

First things first, I downloaded Bootsrap from the official website. I used Bootsrap because I believed that it would make my life easier when using CSS.
I then decided to follow a Bootstrap tutorial on YouTube. Although Bootstrap has documentation on how to use its features, I wanted to supplement my learning with a tutorial. (The tutorial is extremely helpful and I recommend everyone to use it when learning Bootstrap.)
Since the tutorial guy (who I will call tutor for the rest of this documentation) recommended to use a plugin called Emmet, I decided to download it. Emmet is extremely helpful because it autofills lines of code for you (for example, instead of writing all the doctypes and html tags yourself, just type in a "!" and Emmet will do it for you). The plugin definetly sped up my web developing process and I highly recommend you all to use it. (NOTE: Those who want to use Emmet on Sublime Text will need to use this link instead of the link provided by the Emmet website.) For all the website's, I used Bootstrap's own CSS and Javascript plugins.


Step 2: Recreate The Main Page

Second, I wanted to recreate the main page to take away the 1990s look from it.

To start out, I created some containers for the links. I realized they looked ugly, so I used the tutors styling and the container became blue. I then colored the links white to make them noticeable on the container. I repeated these steps on the about me link and the digital fabrication class link. I repeated the process on the links for the Digital Fabrication Class. I also changed the website name from "Ali's Crafts" to "Ali's Portfolio" because I believed it sounded more mature and cleaner.

Step 3: Recreate The About Me Page

The About Me Page frankly looks like something a five year old would have wrote. I don't even talk about myself at all! I changed this by adding images and more information about myself, such as my favorite piano piece to play, and other hobbies I enjoy. I may add a picture of myself later once I find a good photo.

Step 4: Add A Page Talking About My Reasoning In Making This Portfolio

Obviously, the reason I made this portfolio was so I don't get an F in my Digital Fabrication Class, but most people would not know that, so I added an entire page just to say that! Of course, I also talked about my goals with this portfolio and what mechanisms I will be creating.

Step 4: Add A Navigation Bar

I quickly realized that having to constantly use the back arrow to go back to the previous page was annoying, even more annoying for mobile users. So, I decided to add a mobile friendly Navbar by taking the Navbar code from the Bootstrap website and implementing it into EVERY single one of my webpages. (This took thirty minutes to do.) The only thing I took out from the code was the dropdown, disable button, and the search bar; the searchbar in the projects webpage still exists because I wanted to make a way to search and access specific projects quickly. (The search bar doesn't work right now and I have no intention of making it work until I add in more projects.) The Digital Fabrication pages have their own unique DFab Home button to quickly go back to the DFab Main Page.

Step 5: Make Images Take Up A Screen's Exact Space

The main problem keeping this website from being mobile friendly (apart from the code showcases) is the image size. The images take up a fized amount of space, so if their is not enough space on a screen, mobile users will need to scroll sideways to access full image. To fix this, I edited all images by adding an "img-fluid" class and that fixed the problem, but it added a new problem: some images and videos are too big on PC (It takes up the full screen length). I do not think there is a proper way to fix this so I may have to just deal with it. If I find a solution, I will fix it as quickly as possible. (I will admit, it gives a cool cinematic feel though.)

And that is all of the major changes I did to my website! I may do small updates later, but I intend on making a big website update again in a year (if needed). Feel free to use the new Navbar to navigate around the website!
